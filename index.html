Display : Flex(Child Property)Flex-Grow

  • The flex-grow CSS property sets the flex grow factor, which specifies how much of the flex container's positive free space, if any, should be assigned to the flex item's main size.
  • When the flex container's main size is larger than the combined main sizes of all the flex items, the positive free space can be distributed among the flex items, with each item's growth being their grow factor values as a proportion of the sum total of all the flex items' grow factors.
